# Requirements

- PHP 8+
- PHP mbstring extension installed and loaded.
- Composer 2+


# Installation

On terminal type:

```shell
composer require torugo/util
```

Or add to your require list on composer.json file:

```json
{
    "require": {
        "torugo/util": "^1.0.0"
    }
}
```
